# 그룹 기능
1. 큐시트 리스트 공유
    개인 큐시트와 별개.
    그룹별로 관리
2. 그룹 멤버 관리
    초대받은 사람만 그룹 큐시트 접근
3. 그룹별 공연 큐시트 관리
    공연마다 큐시트 목록을 분리
4. 그룹 메모
    합주 준비물, 역할, 전달사항 정리
5. 그룹 메시지
    초대, 공지, 수정 요청 확인

# 현재 구현된 기능

## 메인
1. 큐시트 만들기
2. 큐 목록 확인
3. 전체 공연 시간 자동 계산
4. 관객용 목록으로 이동
5. 회원가입 페이지로 이동

## 로그인
1. 이메일 + 비밀번호 로그인
2. Google 로그인 준비
3. 로그인 세션 확인
4. 로그아웃
5. 비밀번호 미등록 계정 안내

## 회원가입
1. 이메일 아이디 입력
2. 이메일 인증코드 발송
3. 이메일 인증코드 확인
4. 개인정보 동의
5. 이름, 생년월일, 메모 입력
6. 휴대폰번호 입력
7. 비밀번호 조건 확인
8. 가입하기
9. 가입 완료 모달 표시
10. 메인 페이지로 이동

## 큐시트
1. 곡 추가
2. 인터미션 / 구분선 추가
3. 곡 제목 입력
4. BPM 입력
5. 소요 시간 입력
6. 통기타 튜닝 입력
7. 베이스 튜닝 입력
8. 메모 입력
9. 순서 변경
10. 삭제
11. 전체 삭제
12. 목록 저장
13. 메트로놈 실행

## 관객용 큐시트
1. 오늘의 큐시트 표시
2. 곡 순서 표시
3. 전체 곡 수 표시
4. 총 소요 시간 표시

## 연습 캘린더
1. 날짜별 연습 시간 기록
2. 연습 메모 작성
3. 날짜별 합계 확인
4. 월별 연습 기록 확인
5. 연습 기록 삭제

## 할 일
1. 공연 준비 할 일 작성
2. 체크리스트 관리
3. 항목 순서 변경
4. 편집 비밀번호 확인
5. DB 저장

## 앨범 아트
1. 곡 제목 기준 앨범 이미지 검색
2. 커스텀 앨범 아트 매칭
3. 이미지 프록시 로딩

# API
1. /api/cues
    큐시트 저장 / 불러오기
2. /api/audience-cues
    관객용 큐시트 조회
3. /api/practice
    연습 기록 저장 / 불러오기
4. /api/todos
    할 일 문서 저장 / 불러오기
5. /api/auth/*
    세션, 로그인, 로그아웃, 회원가입, 이메일 인증, Google 로그인
6. /api/album-art
    앨범 아트 검색
7. /api/album-image
    앨범 이미지 프록시
