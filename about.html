Why Cue Sheet

공연 준비는 곡 목록 하나로 끝나지 않으니까.

합주실에서 정한 내용이 공연 직전까지 이어지도록, 자주 놓치는 정보를 같은 흐름 안에 모았습니다.

01

기억 대신 기록으로

BPM, 튜닝, 멘트와 전환 시간을 곡 옆에 남겨 다음 합주에서도 바로 이어갑니다.

02

개인 준비와 팀 준비를 연결

내 연습 기록과 할 일을 관리하고, 완성한 목록은 그룹 큐시트와 공연에 연결합니다.

03

무대 위 흐름까지 선명하게

곡과 인터미션의 순서를 조정하며 총 시간을 확인하고, 공개된 순서는 관객 화면으로 보여 줍니다.

Core Features

공연 전 과정을 위한 여섯 가지 도구

각 기능은 따로 쓸 수 있고, 로그인하면 계정과 그룹을 기준으로 준비 흐름을 이어갈 수 있습니다.

One Connected Flow

Set에서 Show까지, 네 단계면 충분합니다.

공연 준비의 현재 위치를 놓치지 않도록 각 단계가 다음 작업으로 자연스럽게 이어집니다.

  1. 01
    Set

    큐시트를 만듭니다

    곡, BPM, 시간과 튜닝을 입력해 공연의 뼈대를 잡습니다.

  2. 02
    Practice

    연습과 준비를 기록합니다

    날짜별 연습량과 남은 할 일을 확인하며 완성도를 높입니다.

  3. 03
    Team

    그룹과 공유합니다

    멤버를 초대하고 그룹 큐시트, 공지와 공연 정보를 연결합니다.

  4. 04
    Show

    무대 흐름을 확인합니다

    최종 순서를 관객용 화면으로 열어 공연 당일까지 함께 봅니다.

Built for everyone

공연을 만드는 사람 모두에게

같은 공연을 준비해도 필요한 정보는 다릅니다. 역할마다 꼭 필요한 화면에 빠르게 닿을 수 있게 구성했습니다.

LEADER

밴드 리더

세트리스트와 공연 정보를 정리하고, 그룹 멤버에게 큐시트와 공지를 전달합니다.

  • 그룹 생성·멤버 초대
  • 그룹 큐시트·공연 연결
  • 공지와 준비 흐름 관리
MEMBER

밴드 멤버

내 파트에 필요한 세팅을 확인하고, 개인 연습 기록과 팀의 최신 목록을 함께 봅니다.

  • BPM·튜닝 빠른 확인
  • 연습 기록·개인 메모
  • 그룹 초대와 공지 확인
AUDIENCE

공연 관객

편집 도구 없이도 현재 공개된 곡 순서를 보기 좋은 목록으로 간편하게 확인합니다.

  • 공연 순서 한눈에 보기
  • 곡과 파트 정보 확인
  • 이미지 저장·링크 복사
